## 📱 ایپ کیسے انسٹال کریں؟

### Android پر:
1. APK فائل ڈاؤن لوڈ کریں
2. **Settings** → **Security** → **Unknown Sources** کو آن کریں
3. APK فائل کھولیں
4. **Install** بٹن دبائیں

## 🤝 شراکت (Contribution)
اگر آپ اس پروجیکٹ میں بہتری لانا چاہتے ہیں:
1. اس ریپوزٹری کو Fork کریں
2. اپنی تبدیلیاں کریں
3. Pull Request بھیجیں
